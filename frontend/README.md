# Frontend (MVP)

Базовый интерфейс для выбора офиса/этажа, просмотра доступности мест и создания брони.

## Запуск

1. Запустить backend (см. `../backend/README.md`).
2. Запустить статический сервер:

```bash
python -m http.server 5173
```

3. Открыть в браузере `http://localhost:5173/frontend/`.

## Возможности

- выбор офиса и этажа;
- проверка доступности мест на заданную дату и слот;
- создание бронирования;
- отображение типа места, зоны и закрепления.
- просмотр правил бронирования для выбранного офиса;
- отображение плана этажа по ссылке `plan_url`.
